# Hey, ich bin Sascha 👋

Java-Entwickler mit Fokus auf Backend- und Server-Architektur. Seit 2016 entwickle ich
Plugins und Systeme im Minecraft/Spigot-Ökosystem.
Von einzelnen Gameplay-Features bis zu
Netzwerk-Architekturen für mehrere tausend gleichzeitige Spieler.

## Was ich mache

- Plugin- und Server-Entwicklung mit der Spigot/Bukkit-API seit 2016
- Backend-Systeme mit Fokus auf Performance, Nebenläufigkeit und Skalierbarkeit
- Datenhaltung für latenzkritische, verteilte Anwendungen
